# Jamf Pro

Integrate Jamf Pro with Quilr to enhance visibility into your organization's device management and identity security posture, ensuring comprehensive protection and streamlined access management.

- **Category:** Mobile Device Management (MDM)
- **Integration Type:** OAuth 2.0 / API
- **Vendor:** Apple

## Setup

1. Go to **Integrations** and open the **Available** tab.
2. Click **+ Add** on the **Jamf Pro** tile.
3. Enter your Jamf Pro server URL and API credentials with the required access.
4. Click **Allow** to authorize the connection.

## Required Permissions

The Jamf Pro integration requires API access to:

- Computer inventory and details
- Mobile device inventory
- User information
- Device groups and policies
- Configuration profiles

## What This Integration Does

- Monitors Apple device inventory (Mac, iPhone, iPad).
- Tracks device compliance status.
- Identifies unmanaged or non-compliant devices.
- Provides visibility into macOS/iOS security posture.
