# ChatGPT

Integrate ChatGPT with Quilr to monitor OpenAI API usage and AI assets.

- **Category:** Asset Discovery
- **Integration Type:** API Key
- **Vendor:** OpenAI

## Setup

1. Go to **Integrations** and open the **Available** tab.
2. Click **+ Add** on the **ChatGPT** tile.
3. Enter your OpenAI API key with the required read access.
4. Click **Allow** to authorize the connection.

## Required Permissions

- OpenAI API key with read access
- Organization-level access (for enterprise features)

## What This Integration Does

- Monitors ChatGPT/OpenAI API usage.
- Tracks AI model deployments.
- Discovers AI assets using OpenAI services.
