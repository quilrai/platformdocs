# Ollama

Integrate Ollama with Quilr to discover and monitor locally-hosted LLM deployments running on Ollama.

- **Category:** Asset Discovery
- **Integration Type:** API

## Setup

1. Go to **Integrations** and open the **Available** tab.
2. Click **+ Add** on the **Ollama** tile.
3. Enter your Ollama server endpoint and API authentication credentials (if configured).
4. Click **Allow** to authorize the connection.

## Required Permissions

- Ollama server endpoint access
- API authentication credentials (if configured)

## What This Integration Does

- Discovers locally-hosted LLM instances running on Ollama.
- Monitors Ollama model deployments.
- Tracks AI/ML asset inventory for self-hosted models.
- Provides visibility into local AI infrastructure usage.
