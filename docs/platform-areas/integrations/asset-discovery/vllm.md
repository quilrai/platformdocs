# vLLM

Integrate vLLM with Quilr to discover and monitor self-hosted LLM deployments.

- **Category:** Asset Discovery
- **Integration Type:** API

## Setup

1. Go to **Integrations** and open the **Available** tab.
2. Click **+ Add** on the **vLLM** tile.
3. Enter your vLLM server endpoint and API authentication credentials.
4. Click **Allow** to authorize the connection.

## Required Permissions

- vLLM server endpoint access
- API authentication credentials

## What This Integration Does

- Discovers self-hosted LLM instances.
- Monitors vLLM server deployments.
- Tracks AI/ML asset inventory.
