# Ping Identity

Integrate Ping Identity with Quilr to enhance visibility into your organization's identity and access management security posture, ensuring comprehensive protection and streamlined authentication workflows.

- **Category:** Identity
- **Integration Type:** OAuth 2.0 / API

## Setup

1. Go to **Integrations** and open the **Available** tab.
2. Click **+ Add** on the **Ping Identity** tile.
3. Enter your Ping Identity environment details and credentials with the required access.
4. Click **Allow** to authorize the connection.

## Required Permissions

The Ping Identity integration requires access to:

- User directory information
- Authentication events
- Application configurations
- Policy settings

## What This Integration Does

- Discovers and monitors user directory information.
- Reviews application configurations and policy settings.
- Analyzes authentication events for security insights.
- Surfaces identity and access management security posture.
