# Okta

Integrate Okta with Quilr to quickly gain clear visibility into your organization's identity security posture, ensuring strong protection.

- **Category:** Identity
- **Integration Type:** OAuth 2.0 / API Token

## Setup

1. Go to **Integrations** and open the **Available** tab.
2. Click **+ Add** on the **Okta** tile.
3. Enter your Okta domain and an API token with the required read access.
4. Click **Allow** to authorize the connection.

## Required Permissions

The Okta integration requires an API token with read access to:

- Users and user profiles
- Groups and group memberships
- Applications and app assignments
- Authentication policies
- System logs and audit events

## What This Integration Does

- Discovers and monitors Okta users, groups, and applications.
- Reviews authentication policies and app assignments.
- Analyzes system logs and audit events for security insights.
- Surfaces identity security posture across your Okta tenant.
