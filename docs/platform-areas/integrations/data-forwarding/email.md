# Email

Integrate email with Quilr to send security alerts via email.

- **Category:** Reporting
- **Integration Type:** Custom

## Setup

1. Go to **Integrations** and open the **Available** tab.
2. Click **+ Add** on the **Email** tile.
3. Configure the delivery settings below.
4. Click **Allow**.

## Required Configuration

- Email recipients
- Alert types

## What This Integration Does

- Sends email notifications for security alerts.
- Enables scheduled reports.
- Provides customizable alert templates.
