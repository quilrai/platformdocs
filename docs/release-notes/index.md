# Release Notes

  detail drawers.
- Visible sources depend on tenant configuration and permissions for [Browser Extension](/platform-areas/browser-extension),
  [Endpoint Agent](/platform-areas/endpoint-agent), [AI Gateway](/platform-areas/ai-gateway),
  [MCP Gateway](/platform-areas/mcp-gateway), and Compliance integrations.
